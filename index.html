About

大学院で音楽情報処理に関する研究を行っています。 特に、楽曲同士の「どの部分が似ているのか」を可視化し、 ユーザが聴き比べながら音楽の類似性を理解できるシステムの開発に取り組んでいます。

また、研究だけでなく、シンセサイザやメロディ生成プログラムなど、 音楽とソフトウェアを組み合わせた制作にも取り組んでいます。 将来的には、ユーザにとって使いやすいシステムを設計・実装できるエンジニアを目指しています。

Research

音楽類似度理解支援システムの研究

楽曲同士の類似性は感覚的に語られることが多く、 「どの部分が似ているのか」を説明することは容易ではありません。 そこで、複数の楽曲間の類似区間を視覚的・聴覚的に提示し、 ユーザが楽曲の特徴を理解しやすくする支援システムについて研究しています。

研究で取り組んでいること

  • 楽曲音声からの特徴抽出
  • 2曲間の時系列類似度マップの生成
  • 類似区間の可視化と音声再生UIの実装
  • ユーザが直感的に理解できるインタフェース設計
  • 音楽情報処理を用いた理解支援システムの検討
Music Information Retrieval Information Visualization Human Interface Python

Works

音楽類似度マップ可視化アプリ

2曲間の類似度を時系列マップとして表示するPythonアプリです。 ユーザはマップ上の任意の範囲を選択し、 選択した区間に対応する2曲の音声を再生できます。 類似している部分を視覚的・聴覚的に確認できるようにしました。

工夫した点

  • 類似度マップと音声再生を連動させた点
  • 選択範囲内に再生バーを表示し、現在再生位置を分かりやすくした点
  • 複数の音響特徴量を比較できるようにした点
  • 大きな類似度行列でも扱いやすいように処理を分割した点
  • 研究用ツールとして、実験・確認を行いやすいUIにした点
Python Pygame librosa Audio Processing Visualization

ポリフォニックシンセサイザ

Pythonで開発したソフトウェアシンセサイザです。 キーボード入力によるリアルタイム演奏に対応し、 複数音の同時発音、オシレータ、フィルタ、ADSRエンベロープなどを実装しました。

工夫した点

  • 複数音を同時に鳴らせるポリフォニック再生に対応した点
  • ADSRエンベロープにより、音の立ち上がりや余韻を制御できるようにした点
  • ローパスフィルタによって音色変化を表現した点
  • キーボード入力によるリアルタイム演奏を可能にした点
  • 画面上でパラメータを確認しながら調整できるようにした点
Python DSP Synthesizer Real Time Audio UI

自動メロディ生成システム

音楽理論やコード進行をもとに、新しいメロディを自動生成するプログラムです。 生成したメロディはMIDIとして出力でき、DAWでの作曲に活用できるようにしました。

工夫した点

  • コード進行に合う音を優先してメロディを生成するようにした点
  • 不自然な音程の跳躍を抑え、自然な旋律になるようにした点
  • リズムパターンを調整できるようにした点
  • MIDI出力に対応し、作曲ソフトで利用しやすくした点
  • 音楽制作の補助ツールとして使える構成にした点
Python MIDI Music Generation Algorithm Composition Support
